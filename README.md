# Documentario de prensa y calendario de óperas italianas (1747–1758)

Este repositorio contiene la **versión pública de la web** del "Documentario de prensa y calendario de las representaciones de ópera italiana organizadas por Carlo Broschi Farinelli durante el reinado de Fernando VI y Bárbara de Braganza (1747–1758)".

La web se puede consultar en línea aquí:  
[https://dmartinsaez.github.io/opera-gacetas-siglo-XVIII/](https://dmartinsaez.github.io/opera-gacetas-siglo-XVIII/)

---

## Contenido del repositorio público

- `index.html` → Página principal de la web, que incluye buscador, tablas y transcripciones.
- `css/` y `js/` (si existieran) → Archivos de estilo y scripts necesarios para que la web funcione.
- **No incluye los datos originales (`datos.json`)**, que se mantienen privados en otro repositorio para proteger el trabajo de edición y los datos de archivo.

---

## Descripción

Este proyecto permite:

- Consultar transcripciones de noticias sobre óperas realizadas en el Coliseo del Buen Retiro de Madrid entre 1747 y 1758.
- Buscar palabras o fragmentos en todas las noticias, con normalización histórica.
- Ver y ocultar transcripciones individuales de cada noticia.
- Consultar información sobre libretistas, compositores y contexto de las representaciones.
